## 📋 نظرة عامة

**Sky Properties** هو مشروع تجريبي احترافي لمنصة إدارة وتسويق العقارات (PropTech Platform) تم تطويره بواسطة **CyberPath Digital** كجزء من أعمال الشركة. يبرز المشروع قدرات CyberPath Digital في تصميم وتطوير الأنظمة والمنصات الرقمية المتقدمة.

## 🎯 الهدف

إنشاء موقع تجريبي احترافي يعرض إمكانيات منصة عقارية حديثة تشمل:
- إدارة العقارات
- إدارة المستأجرين
- متابعة الصيانة
- تقارير وتحليلات
- لوحات تحكم
- البحث العقاري
- الأتمتة والذكاء الاصطناعي
