# Sky Properties | سكاي بروبرتيز

![Sky Properties](https://via.placeholder.com/1200x400/0A84FF/FFFFFF?text=Sky+Properties)

## 📋 نظرة عامة

**Sky Properties** هو مشروع تجريبي احترافي لمنصة إدارة وتسويق العقارات (PropTech Platform) تم تطويره بواسطة **CyberPath Digital** كجزء من أعمال الشركة. يبرز المشروع قدرات CyberPath Digital في تصميم وتطوير الأنظمة والمنصات الرقمية المتقدمة.

## 🎯 الهدف

إنشاء موقع تجريبي احترافي يعرض إمكانيات منصة عقارية حديثة تشمل:
- إدارة العقارات
- إدارة المستأجرين
- متابعة الصيانة
- تقارير وتحليلات
- لوحات تحكم
- البحث العقاري
- الأتمتة والذكاء الاصطناعي

## الخطوة 3: رفع المشروع على GitHub

### 1. إنشاء مستودع جديد على GitHub

1. اذهب إلى [GitHub](https://github.com)
2. سجل الدخول إلى حسابك
3. انقر على زر **New repository** (أخضر)
4. أدخل اسم المستودع: `sky-properties`
5. اختر **Public** أو **Private**
6. لا تقم بتفعيل README أو .gitignore (سنضيفها لاحقاً)
7. انقر على **Create repository**

### 2. رفع الملفات إلى GitHub

افتح Terminal (Mac/Linux) أو Command Prompt (Windows) ونفذ الأوامر التالية:

```bash
# 1. انتقل إلى مجلد المشروع
cd path/to/sky-properties

# 2. تهيئة Git
git init

# 3. إضافة جميع الملفات
git add .

# 4. إنشاء commit أول
git commit -m "Initial commit: Sky Properties portfolio project"

# 5. إضافة remote repository
git remote add origin https://github.com/yourusername/sky-properties.git

# 6. رفع الملفات
git push -u origin main
إذا كان المستودع يستخدم master بدلاً من main:

bash
git push -u origin master
